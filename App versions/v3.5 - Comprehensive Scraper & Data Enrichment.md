## Scraper Overhaul

### Eventbrite Fixes
- **Root cause of missing events**: artist name was extracted from URL slug (`eric-clapton-live-at-g-live-tickets-1234` → "Eric Clapton Live At G Live"), which failed to match "Eric Clapton" in the DB
- **Fix**: new `extractArtistFromTitle()` helper uses the event title first (strips tour names, venue suffixes, "ft." etc.) and falls back to URL slug
- **Added 32 UK city-specific pages**: Guildford, all major UK cities — events like Eric Clapton at G Live will now appear in the city feed even if they don't rank on the national browse

### Venue-Direct Scraping (per venue batch)
Every 6-hour run now processes 50 venues, each querying **4 sources** instead of 2:
1. **Skiddle** (was already working)
2. **Ticketmaster** — via stored `tmVenueId` field (see enrichment script below)
3. **Eventbrite** — city-specific search filtered to the venue
4. **Dice.fm** — venue name search
5. **Venue website** JSON-LD (was already working)

### Other Source Improvements
- **TM bulk scan**: throttle reduced from 22h → 6h (every scraper cycle)
- **Songkick metro areas**: expanded from 1 city (London) to 16: Manchester, Birmingham, Glasgow, Leeds, Bristol, Edinburgh, Liverpool, Newcastle, Sheffield, Nottingham, Cardiff, Brighton, Oxford, Cambridge, Guildford

## Enrichment Scripts (run on-demand)

### `scripts/enrich-venues-tm-ids.cjs`
Queries the Ticketmaster Discovery Venues API for every venue in DynamoDB and stores the `tmVenueId`. This unlocks venue-direct TM queries in every scraper run.

```bash
node scripts/enrich-venues-tm-ids.cjs
# Optional: node scripts/enrich-venues-tm-ids.cjs BATCH_SIZE=1000
```

### `scripts/enrich-venues-web.cjs`
Adds to each venue: `bio` (Wikipedia), `imageUrl`, `address` (MusicBrainz / Nominatim), `lat/lng`, `wikiUrl`.

```bash
node scripts/enrich-venues-web.cjs --limit=500
```

### `scripts/enrich-artists-web.cjs`
Adds to each artist: `bio` (Wikipedia / Last.fm), `genres`, `imageUrl`, `mbid`, `wikiUrl`.

```bash
node scripts/enrich-artists-web.cjs --limit=500 --missing-only
```

### `scripts/discover-new-venues.cjs`
Discovers venues not yet in the DB from MusicBrainz UK place database, Skiddle active venues, and Songkick metro pages.

```bash
node scripts/discover-new-venues.cjs --source=all
node scripts/discover-new-venues.cjs --source=mb      # MusicBrainz only
node scripts/discover-new-venues.cjs --source=skiddle # Skiddle only
```
