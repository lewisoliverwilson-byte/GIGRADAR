## Enrichment Scripts (v3.4)

### `scripts/enrich-artists-lastfm.cjs`
- Queries Last.fm `artist.getInfo` for every artist in DB
- Writes: `genres[]`, `bio`, `lastfmListeners`, `lastfmMbid`
- Rate: 5 req/sec (free tier)
- Est. runtime: ~1 hour for 18,866 artists

### `scripts/enrich-artists-spotify.cjs`
- Uses Spotify Client Credentials (no user login required)
- Searches each artist by name, finds best match
- Writes: `spotify` (URL), `monthlyListeners` (Spotify followers), `genres[]` (if not already set)
- Rate: ~10 req/sec
- Est. runtime: ~30 mins for 18,866 artists

### `scripts/enrich-artists-images.cjs` (already running)
- Deezer → MusicBrainz/fanart.tv fallback
- Target: fill `imageUrl` for all artists without one
