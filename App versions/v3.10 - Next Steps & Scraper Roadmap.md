## Known Limitations & Gaps

### Bandsintown
API returns 403 — public access revoked. No workaround available.

### Eventbrite
Discovery API deprecated in 2023. HTML pages only show 8 events per page (unusable at scale). No viable scraping path.

### Wegottickets / Seetickets
Both Cloudflare-protected or client-side rendered. Not scrapable.

### Facebook Events (~40–60% of grassroots gigs)
Requires per-venue OAuth — see `v4.0 Feature Roadmap.md`.

### Instagram
Requires per-venue OAuth — see `v4.0 Feature Roadmap.md`.
