# v3.9 — Ticketmaster & Songkick Scrapers

**Date:** 2026-04-20  
**Commit:** 2033115

## What Changed

### New Scrapers

#### `scripts/scrape-ticketmaster.cjs`
Uses the Ticketmaster Discovery API v2 to bulk-fetch all UK Music events.
- Splits 14-month window into weekly ranges (to stay under TM's 1,000-result cap per query)
- 5 pages × 200 events = 1,000 events/week max
- `PutCommand` with `tm-{eventId}` gigIds for clean deduplication
- Artist from `_embedded.attractions[0]`, support acts from remaining attractions
- Price ranges extracted as `£X–£Y` strings
- `--resume` support via progress file
- **10,467 gigs added** across 61 weeks (March 2026 – May 2027)

#### `scripts/scrape-songkick.cjs`
Scrapes Songkick metro-area calendar pages using `LD+JSON` MusicEvent schema data.
- No API key — pure HTML schema scraping
- 36 verified UK metro area IDs (discovered via redirect scan of IDs 24350–25200)
- Correct concert URL regex: `/(?:concerts|id)/(\d+)/` (festivals use `/id/`, regular concerts use `/concerts/`)
- 500ms delay between pages; `--resume` support
- **London alone: ~4,150 gigs** per run; full UK run in progress

### New Enrichment Scripts

#### `scripts/enrich-artists-lastfm.cjs`
Enriches all artists with Last.fm `artist.getInfo`:
- Adds: genres (tags), bio summary, listener count, MusicBrainz ID
- 60% hit rate across 26,085 artists → ~15,600 artists enriched with genres/bios
- API key hardcoded: `e2c0791c809dd2a81adde0158dd70c41`

#### `scripts/enrich-artists-images.cjs`
Fetches artist profile images from Deezer and MusicBrainz:
- Deezer: ~35% hit rate, instant, no key
- MusicBrainz Cover Art Archive: 1 req/sec fallback
- Targets artists with no `imageUrl` only

#### `scripts/enrich-venues-musicbrainz.cjs`
Queries MusicBrainz for UK venue places to get official website URLs and capacity.
- 1 req/sec rate limit respected
- Targets venues without a website URL

#### `scripts/enrich-venues-ticketmaster.cjs`
Enriches venues that were seeded by the TM gig scraper using TM's venue endpoint:
- Scans `gigradar-gigs` for `tm-` prefixed gigIds to find TM venue IDs
- Fetches venue images, website, social links, address, capacity
- 953 unique TM venues enriched

#### `scripts/count-db.cjs`
Quick helper to count records in all three DynamoDB tables.
